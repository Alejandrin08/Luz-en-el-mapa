# Luz en el mapa

Proyecto React + Vite (JSX puro) con `react-router-dom` y Tailwind CSS v4.

## Scripts

```bash
npm install
npm run dev      # entorno de desarrollo
npm run build    # build de producción
npm run preview  # preview del build
```

## Estructura

```
src/
  main.jsx              # entrada + router
  styles.css            # tokens del sistema (Tailwind v4)
  components/
    SiteLayout.jsx      # header + footer compartidos
  pages/
    Index.jsx           # /
    Configura.jsx       # /configura
    Historia.jsx        # /historia
    Nacional.jsx        # /nacional
    Recursos.jsx        # /recursos
  assets/
    classroom.jpg
```
